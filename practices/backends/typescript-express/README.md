# typescript-express

## ツールチェーン

- プログラム言語: TypeScript
- パッケージ管理: npm
- Web(バックエンド)フレームワーク: Express.js (+ Node.js)
- linter/formatter: ESLint, Prettier
- テスト: jest, supertest

## セットアップの流れ

### npm プロジェクト

```bash
brew install node
node -v
npm -v

npm init

npm install typescript typescript-eslint -D
npx tsc --init
# 以下略
```

## express.js

```bash
npm install express
# 以下略
```

- [公式ドキュメント](https://expressjs.com/ja/starter/installing.html)
