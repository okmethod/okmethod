# t3-stack

## ツールチェーン

- プログラム言語: TypeScript
- パッケージ管理: npm
- CSS フレームワーク: TailwindCSS
- RPC フレームワーク: tRPC
- Web フレームワーク: Next.js

## セットアップの流れ

### node

```bash
brew install node
node -v
npm -v
```

### Create T3 App

```bash
npm create t3-app@latest
```

- [公式ドキュメント](https://create.t3.gg/en/introduction)
